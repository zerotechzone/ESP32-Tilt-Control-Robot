# ESP32 Tilt Controlled Robot 🚗

This project uses two ESP32-C3 boards:

## 🔹 Project 1 (Controller)
- ESP32-C3 + ADXL345
- Detects tilt and sends commands using ESP-NOW

## 🔹 Project 2 (Robot)
- ESP32-C3 + L298N + Motors
- Receives commands and moves accordingly

## 🎮 Controls
- Tilt Forward → Left
- Tilt Backward → Right
- Tilt Right → Forward
- Tilt Left → Backward

## 📡 Communication
- ESP-NOW (no WiFi router needed)

## ⚡ Features
- Wireless control
- PWM motor speed control
- Real-time response

## 🛠️ Hardware
- ESP32-C3 (2x)
- ADXL345
- L298N Motor Driver
- DC Motors
